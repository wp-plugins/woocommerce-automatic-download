Enables Digital products to download automatically on shop's thank you page.

== Description ==

WooCommerce Automatic Download is a WordPress plugin that allows you to enable automatic download for downloadable products. Once your customers purchase any downloadable product and lands on thank page, the downloadable files will be downloaded automatically. This is really nice feature for any digital content selling websites.

== Installation ==

1. Upload zip to plugins directory or upload it from wp admin panel
2. Activate the plugin
3. No extra configurations required to enable automatic downloads

== Frequently asked questions ==

How to Disable Automatic Downloads?

1. Go to WooCommerce -> Settings -> Products -> Downloadable Products
2. Uncheck Automatic Download checkbox
